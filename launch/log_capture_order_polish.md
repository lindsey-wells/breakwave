# BreakWave — BW-72A Log Capture Order Polish

## Purpose

BW-72A makes the Log tab prioritize fast truth capture before pattern review.

## Changes

- Moves the manual log form above Recent Entries.
- Keeps Recent Entries available below the save flow.
- Adds safer bottom padding so controls are not hidden behind the bottom nav.
- Adds edit auto-scroll so tapping Edit on a recent entry returns the user to the draft form.
- Does not change the log data model or storage format.

## Product rule

Log should help the user capture what happened quickly. Pattern review matters, but it should not block new logging.
