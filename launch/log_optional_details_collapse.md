# BreakWave — BW-72C Log Optional Details Collapse

## Purpose

BW-72C keeps Log fast by making deeper reflection optional.

## Changes

- Keeps Entry Type, Intensity, Trigger Signals, and Healthy Replacement Action visible.
- Moves Thought, Action Taken, Consequence, and Better Plan into Add reflection details.
- Changes Notes into Optional notes.
- Keeps all existing fields, controllers, and local storage behavior.
- Keeps Recent Entries compact.

## Product rule

Log should support fast truth capture first, then deeper reflection when the user has capacity.
