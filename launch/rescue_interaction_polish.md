# BreakWave — BW-71A Rescue Interaction Polish

## Purpose

BW-71A makes Rescue feel more active, honest, and useful during an urge.

## Changes

- Moves Next Right Action higher so the user chooses a clean redirect earlier.
- Adds an animated wave progress visual to the 90-second Wave Timer.
- Converts Calm Reset from SnackBar-only feedback into a guided breathing stepper.
- Adds honest Rescue completion outcomes:
  - I made it through
  - Still strong
  - I slipped
- Saves Victory, Urge, and Slip outcomes from Rescue.
- Opens Support after a logged slip so the user has an immediate next support path.

## Product rule

Rescue should not only celebrate wins. It should help users tell the truth quickly, interrupt the pattern, and keep moving without shame.
