# BreakWave — BW-73A Support Declutter Groups

## Purpose

BW-73A makes Support feel organized without adding a hamburger menu or new navigation system.

## Changes

- Keeps Support inside the existing bottom navigation.
- Replaces one long stack with collapsible Support groups.
- Keeps urgent support expanded by default.
- Groups recovery setup, privacy, learning, Plus, contact, and advanced tools.
- Preserves all existing Support tools and actions.

## Product rule

Support should feel like an organized harbor, not a storage closet.
