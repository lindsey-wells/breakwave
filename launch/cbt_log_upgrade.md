# BreakWave — BW-63 CBT-Informed Log Upgrade

## Purpose

BW-63 upgrades the Log tab from simple tracking into a lightweight CBT-informed behavior-change record.

## Core model

Trigger → Thought → Urge → Action → Consequence → Better Plan

## Added fields

- Thought before the urge
- Healthy replacement action
- Action taken
- Consequence / what happened next
- Better plan for next time

## Product principle

This is not therapy, diagnosis, or clinical treatment. It is a support tool that helps users notice patterns, interrupt automatic behavior, and build healthier responses one moment at a time.

## MVP constraint

Keep the log lightweight. The goal is fast honesty, not a long therapy worksheet.
