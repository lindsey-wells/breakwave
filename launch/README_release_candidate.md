# BreakWave Release Candidate Sweep

This folder contains the BW-36 launch-trust sweep materials.

## Files
- `release_candidate_smoke_checklist.md` — manual smoke checklist
- `release_candidate_blockers.md` — only real launch blockers go here

## How to use this pass
1. Run the checklist in order
2. Log only real blockers
3. Defer non-blockers unless they affect trust or save reliability
4. Keep feature scope closed during the sweep

## Priority standard
BreakWave launch trust depends most on:
- rescue flow reliability
- save flow reliability
- Home / Log refresh trust
- premium gate correctness
- privacy / reminder resilience
