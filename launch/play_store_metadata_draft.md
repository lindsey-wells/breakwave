# BreakWave — Play Store Metadata Draft

## App name
BreakWave

## Short description
Ride the wave. Interrupt the pattern. Recover with calm, practical support.

## Long description
BreakWave is a recovery app built for the moment the wave starts rising.

It is designed to help you interrupt the pattern earlier, stay honest, and take the next right step when urges, slips, secrecy, or late-night drift start getting louder.

BreakWave includes:
- fast urge entry from Home
- Rescue tools for real-time interruption
- wave timer support
- relapse/slip logging
- daily check-ins
- practical insights
- support contact tools
- reminders and risky-time nudges
- bedtime danger mode
- privacy controls

BreakWave supports two recovery paths:
- Secular: calm, practical, direct support without religious language
- Christian: prayer, Scripture, grace-forward truth, and explicitly Christian recovery language

BreakWave uses a hybrid model:
- core immediate help stays free
- BreakWave Plus is for deeper transformation tools like premium rescue content, deeper insights, accountability depth, guided routines, Christian depth packs, exports, and future advanced tools

BreakWave is built to help in the next 2–5 minutes when the wave is rising — not just track what happened after the fact.
