# BreakWave — BW-72B Log Declutter + Other Capture

## Purpose

BW-72B makes the Log tab faster, clearer, and more flexible without changing the local log data model.

## Changes

- Shortens Entry Type copy.
- Shortens Intensity copy.
- Adds Other trigger capture.
- Adds Other replacement action capture.
- Expands healthy replacement actions with practical options.
- Compacts Recent Entries with expandable details.
- Keeps Edit and Delete available.
- Keeps all log data saved locally on device.

## Product rule

Log should feel like fast truth capture, not recovery homework.
